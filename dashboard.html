Setup Struktur

          
            
            
Technische Felder sind Signalquellen. Das OpenAI/LLM-Rollenteam nutzt diese Daten als Rollen-Prompts; Chart-Farben bleiben in der Chart-Ansicht.

          

        

        

          
BOS / CHoCH

          
            
            

            
Wicks: Hoch/Tief reicht für Break. Candle Close: Kerzenschluss muss das Swing-Level brechen.

            
Begrenzt, wie viele Tage rückwirkend BOS / CHoCH Linien angezeigt und an den Market Structure Analyst gesendet werden. 0 = gesamte geladene Historie.

            

              
Farbe für steigende BOS / CHoCH Linien.

              
Farbe für fallende BOS / CHoCH Linien.

            

          

        

        

          
LL / HH Box

          
            
            

            
Anzahl Kerzen für neue Hoch-/Tief-Bereiche. Höher = weniger, aber größere Strukturboxen.

            
Verlängert HH-/LL-Boxen nach rechts. Minimum 2 Kerzen, damit die Zone sichtbar weiterläuft.

            
Begrenzt Anzeige und Rohdaten für HH-/LL-Boxen. 0 = gesamte geladene Historie.

            

              
Farbe für obere HH-/High-Boxen.

              
Farbe für untere LL-/Low-Boxen.

            

          

        

        

          
HH / LH / HL / LL

          
            
            

            
Kerzen links/rechts zur Pivot-Bestätigung. Höher = weniger Labels, niedriger = mehr Labels.

            
Begrenzt HH, LH, HL, LL Labels im Chart und für die Struktur-Auswertung. 0 = gesamte geladene Historie.

            

              
Farbe für bullische Swing Labels HH und HL.

              
Farbe für bearische Swing Labels LH und LL.

Mindestanzahl technischer Signalquellen, die in dieselbe Richtung zeigen muessen.
